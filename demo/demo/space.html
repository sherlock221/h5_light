空格我们见的应该非常多,但是在网页中 用space产生空格是不会累加的 (只有1个),需要用html实体才会累加.

空格 描述以及说明
&nbsp; 也就是按下space键产生的空格。在HTML中，如果你用空格键产生此空格，空格是不会累加的（只算1个）。要使用html实体表示才可累加
&ensp; 此空格有个相当稳健的特性，就是其占据的宽度正好是1/2个中文宽度，而且基本上不受字体影响。
&emsp; 该空格学名不详。为了便于记忆，我们不妨就叫它"恶魔(e m-o)空格"。此空格也传承空格家族一贯的特性：透明滴！此空格也有个相当稳健的特性，就是其占据的宽度正好是1个中文宽度，而且基本上不受字体影响。

表单应用

还记得原来写表单时候吗,用户名是3个 密码是2个 邮箱4个 字. 我们的做法都是外层设置固定宽度 然后在设置text-align = "left".
现在我们用空格来做

Code

<ul  style="list-style: none;" class="f16">
    <li class="li mt10">用&ensp;户&ensp;名 ：<input type="text" /></li>
    <li class="li mt10">密&emsp;&emsp;码 ：<input type="password" /></li>
    <li class="li mt10">电子邮箱 ：<input type="text" /></li>
</ul>
